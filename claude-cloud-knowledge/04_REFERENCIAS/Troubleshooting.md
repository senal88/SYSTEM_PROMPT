# Troubleshooting

## 1Password
- Verificar: `op-config-check`
- Login: `op-signin-auto`
- Exportar: `op-export-vault.sh`

## GitHub
- Verificar: `gh auth status`
- Setup: `./gh-setup.sh`

## Hugging Face
- Verificar: `hf-status`
- Setup: `./hf-setup.sh`

## Docker
- Status: `docker-compose ps`
- Logs: `docker-compose logs -f`
- Restart: `docker-compose restart`
