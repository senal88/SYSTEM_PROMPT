# README — Raycast Gem Expert

Este repositório contém templates, prompts e workflows profissionais para desenvolvimento de Gems otimizadas para Gemini 2.5 Pro, com integração Raycast e apoio do Perplexity Pro.

## Estrutura

- `prompts/workflow_raycast_gem.md`: Workflow completo, arquitetura e checklist para criação de Gems.
- `prompts/gem_gemini_2.5.md`: Template profissional pronto para uso e customização.

## Como Usar

1. Siga o `workflow_raycast_gem.md` para estruturar sua pesquisa e coleta de conhecimento.
2. Utilize o template `gem_gemini_2.5.md` para criar e documentar sua Gem.
3. Valide e aprimore usando o checklist ao final de cada ciclo.

## Recomendações

- Priorize fontes primárias e documentação oficial.
- Mantenha o conteúdo copy-paste friendly e Raycast-friendly.
- Documente melhorias e aprendizados para futuras versões.
